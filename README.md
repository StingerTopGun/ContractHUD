# FS22 Contract HUD

Helpful small HUD displaying the current completion percentage of all active contracts

![](screenshots/3_small.png?raw=true)

### Hotkeys:
ALT + W  -  Toggle between 3 possible modes (progress bar, percentage [default], hidden)
